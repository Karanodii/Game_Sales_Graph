# 🎮 Video Game Sales Analysis using Tableau

## Overview

This project focuses on analyzing a Video Game Sales dataset using Tableau. The main aim was to explore the data and create an interactive dashboard that makes it easy to understand sales patterns across different platforms, genres, publishers, and regions.

Instead of looking at rows of raw data, the dashboard presents the information through visualizations that help identify trends and compare performance. It provides a clear picture of how the gaming industry has changed over the years and which factors have had the biggest impact on global sales.

Working on this project also helped me improve my understanding of data visualization, dashboard design, and how to present data in a meaningful way.

---

## Project Objectives

The project was created with the following goals:

* Analyze video game sales from different regions.
* Compare the performance of gaming platforms.
* Find the most popular game genres based on sales.
* Identify the leading publishers in the industry.
* Study how game sales have changed over the years.
* Build an interactive dashboard that allows users to explore the data easily.

---

## Tools Used

* Tableau
* CSV Dataset
* Microsoft Excel (for basic data checking and formatting)

---

## Dataset Information

The dataset contains information about video games released on different platforms over multiple years. Each record includes important details that can be used for analysis, such as:

* Game Name
* Platform
* Genre
* Publisher
* Release Year
* North America Sales
* Europe Sales
* Japan Sales
* Other Sales
* Global Sales

These fields make it possible to compare sales across regions and understand the overall performance of different games.

---

## Dashboard Features

The dashboard includes several visualizations that help explore the dataset from different angles.

### Genre Analysis

Shows which genres have generated the highest sales and how they compare with each other.

### Platform Analysis

Compares sales across gaming platforms to identify which platforms performed the best.

### Publisher Analysis

Displays the publishers with the highest total sales and their contribution to the market.

### Regional Sales

Breaks down sales by region, making it easier to compare the popularity of games in North America, Europe, Japan, and other parts of the world.

### Year-wise Sales Trend

Shows how global sales have changed over time and highlights years with higher or lower sales.

### Interactive Filters

Users can filter the dashboard based on different fields such as platform, genre, publisher, and release year to explore the data in more detail.
